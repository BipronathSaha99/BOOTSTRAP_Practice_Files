
# Bootstrap Practice Files

Bootstrap is a CSS framework which makes coding very easier 
and also makes the full responsive website for users.


## Bootstrap Lessons

This Practice files cover the basic to advanced Bootstrap 
Lessons.

- Accordion 
- Alerts
- Badge
- Breadcrumb
- Buttons
- Button group
- Card
- Carousel
- Close button
- Collapse
- Dropdowns
- List group
- Modal
- Navs & tabs
- Navbar
- Offcanvas
- Pagination
- Popovers
- Progress
- Scrollspy
- Spinners
- Toasts
- Tooltips
